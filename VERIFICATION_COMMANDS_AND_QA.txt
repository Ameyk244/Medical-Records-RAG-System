Medical Records RAG Verification Commands And Q/A
=================================================

Purpose
-------
Use this file to verify ingestion, extraction, retrieval, synthesis, and refusal behavior
against every sample dataset currently in this repo:

1. tests/sample_discharge_summary.pdf      -> patient_id P456
2. tests/sample_medical_record_case2.pdf   -> patient_id P982
3. tests/sample_meds.csv                   -> patient_id P123
4. tests/sample_labs.csv                   -> patient_id P123

Notes
-----
- Use Python 3.13 if possible. The repo requirements are pinned for Python 3.13.
- If you are using Python 3.14, `pip install -r requirements.txt` may fail on
  `voyageai==0.3.7`. In that case, install the available package with:
  pip install voyageai greenlet "psycopg[binary]"
- Add real keys to `.env` before full query/synthesis:
  VOYAGE_API_KEY=...
  GOOGLE_API_KEY=...
- If Gemini quota is exhausted, use `--retrieval-only`. Retrieval can still be verified.


1. Environment Setup
--------------------

python3.13 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

If using Python 3.14 and the pinned Voyage package fails:
pip install google-genai voyageai greenlet "psycopg[binary]" sqlalchemy pgvector pdfplumber hl7 python-dotenv

Create local environment file if missing:
cp .env.example .env

Edit `.env` and fill:
GOOGLE_API_KEY=
VOYAGE_API_KEY=
JWT_SECRET=
DB_ENCRYPTION_KEY=

Start local services:
docker-compose up -d

Apply DB migrations:
PYTHONPATH=. alembic upgrade head

11. Quick Pass/Fail Criteria
----------------------------

PASS:
- PDF extraction debug finds patient_name, DOB, age, sex, history, allergies.
- P456 demographic queries retrieve PATIENT DEMOGRAPHICS at rank 1.
- P982 demographic queries retrieve PATIENT DEMOGRAPHICS at rank 1.
- P456 allergy queries retrieve ALLERGIES.
- P982 allergy queries retrieve ALLERGIES.
- P456 PMH/smoking queries retrieve PAST MEDICAL HISTORY.
- P982 PMH/smoking queries retrieve PAST MEDICAL HISTORY.
- P123 med queries retrieve Sample Meds.
- P123 lab queries retrieve Sample Labs.
- Unsupported questions refuse when synthesis is available.

FAIL:
- Demographics query retrieves medication-only chunks first.
- patient_name is missing from final chunk metadata.
- DISCHARGE SUMMARY is classified as medications.
- P456 returns zero chunks after ingest.
- Unsupported questions hallucinate answers not in context.
